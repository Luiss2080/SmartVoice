# Requerimientos del Sistema

Para ejecutar **SmartVoice** correctamente, asegúrate de que tu entorno cumpla con los siguientes requisitos:

## Servidor

-   **PHP**: Versión 8.1 o superior.
-   **Composer**: Gestor de dependencias para PHP.
-   **Base de Datos**: MySQL 5.7+ o MariaDB 10.3+.

## Extensiones de PHP

Asegúrate de tener habilitadas las siguientes extensiones en tu archivo `php.ini`:

-   `BCMath`
-   `Ctype`
-   `Fileinfo`
-   `JSON`
-   `Mbstring`
-   `OpenSSL`
-   `PDO`
-   `Tokenizer`
-   `XML`

## Frontend

-   **Node.js**: Versión 16 o superior (LTS recomendado).
-   **NPM**: Gestor de paquetes de Node.

## Entorno Recomendado

-   **Laragon**: Recomendado para desarrollo local en Windows, ya que incluye Nginx/Apache, MySQL y PHP preconfigurados.
